## Getting Started

### Static Projects (Tasks 1, 2 & 7)
Simply open the HTML files in your browser:
```bash
# Navigate to project folder and open HTML file
cd Task7-Weather_Dashboard  # or Task1-Collapsible-Sidebar or Task2-Contact_Form
# Open index.html or sidebar.html or contact-form.html in browser
```

### React Projects (Tasks 3, 4 & 6)
Install dependencies and run the development server:
```bash
# Navigate to project folder
cd "Task3-Task App"  # or "Task4-BlogHomePage" or "Task6-Dashboard"

# Install dependencies
npm install

# Start development server
npm run dev
```
